Recent Projects

Here are some of the projects I have completed recently. They vary widely from full-stack web applications to a JavaScript game built for the browser. I enjoy working on these projects to practice new skills, sharpen old ones, and watch my code come to life.

OkCoFounder

A site for entrepreneurs in search of business partners, inspired by OkCupid.

  • Matching algorithm calculates and updates “match %” between users in real­time
  • Original design built in HTML and CSS without additional libraries/templates
  • Application data flows, rendering, and data stores architected using React and Flux frameworks

Mars Attacks

A browser-based arcade game built in JavaScript and HTML5 Canvas.

  • Implements object-­oriented JavaScript logic
  • Custom algorithms simulate smooth movement and collision detection for aircraft

ActiveRecord Lite

Custom ORM inspired by the native Rails ORM. It was built using Ruby and guided by an approach of test-­driven development using RSpec.

  • Allows fluid and intuitive access to the database through methods that generate SQL queries (‘where’, ‘find’, ‘all’)
  • Utilizes metaprogramming to generate convenient attr_accessor methods for database table columns
